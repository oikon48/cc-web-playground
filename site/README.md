# Claude Code Sandbox ドキュメントサイト

このディレクトリには、Claude Code on the Webのサンドボックス環境を解説する静的ウェブサイトが含まれています。

## 🎨 デザインコンセプト

Claude Codeをリスペクトしたデザインで、調査結果をわかりやすく解説します。

### カラーパレット
- **Primary**: `#FF6B35` (オレンジ)
- **Secondary**: `#FF8C42` (ライトオレンジ)
- **Accent**: `#FFA366` (アクセント)
- **ダークテーマ**: 黒基調で読みやすく

### デザイン特徴
- 🌗 ダークモード対応
- 📱 レスポンシブデザイン
- ✨ スムーズなアニメーション
- 🎯 直感的なナビゲーション
- 📖 読みやすいタイポグラフィ

## 📂 ファイル構成

```
site/
├── index.html      # メインHTMLファイル
├── style.css       # スタイルシート
```

## 🚀 使い方

### ローカルで表示

1. **シンプルなHTTPサーバーを起動**
   ```bash
   # Python 3を使用
   cd site
   python3 -m http.server 8000
   ```

2. **ブラウザで開く**
   ```
   http://localhost:8000
   ```

### その他の方法

#### Node.jsのhttp-serverを使用
```bash
npx http-server site -p 8000
```

#### PHPを使用
```bash
cd site
php -S localhost:8000
```

## 📋 サイト構成

### セクション一覧

1. **ヒーロー** - プロジェクト概要と主要なスペック
2. **調査概要** - 4つの主要調査項目のカード
3. **重要な発見** - 調査で判明した重要な知見
4. **環境詳細** - システム、リソース、ネットワーク、ツール
5. **ツール・コマンド** - 実行可能なコマンドと制限事項
6. **データ永続化** - 永続化領域とベストプラクティス
7. **ドキュメント** - 詳細な調査レポートへのリンク

## ✨ 機能

### インタラクション

- **スムーズスクロール**: ナビゲーションリンクでページ内を滑らかに移動
- **フェードインアニメーション**: スクロールに応じてコンテンツが表示
- **コードブロックコピー**: コードブロックをクリックでクリップボードにコピー
- **数値カウンターアニメーション**: スタッツが表示時にアニメーション
- **ホバーエフェクト**: カードやボタンに洗練されたホバー効果

### レスポンシブ対応

- デスクトップ（1200px+）
- タブレット（768px - 1199px）
- モバイル（〜767px）

## 🔧 カスタマイズ

### カラーテーマの変更

`style.css`の`:root`セクションでカラー変数を変更できます：

```css
:root {
    --primary: #FF6B35;
    --primary-dark: #E85A2A;
    --primary-light: #FF8C5A;
    /* ... */
}
```

### セクションの追加

`index.html`の`<section>`タグを使用して新しいセクションを追加できます：

```html
<section class="section">
    <div class="container">
        <h2 class="section-title">新しいセクション</h2>
        <!-- コンテンツ -->
    </div>
</section>
```
